# Nature Photography GitHub Pages Starter

## Publish on GitHub Pages

1. Create a repo named `yourusername.github.io` for a user site, or any repo name for a project site.
2. Upload these files to the repo root.
3. In GitHub: **Settings → Pages**.
4. Set the source to **Deploy from a branch**.
5. Choose **main** and **/(root)**.
6. Save.

Your site should publish automatically.

## Why `.nojekyll`?

This tells GitHub Pages not to run Jekyll processing on the site.

## Customize

- Replace the Unsplash image URLs in `index.html` with your own photos.
- Update the site title, about text, and contact links.
- Adjust colors in `styles.css` under `:root`.
